# konka1_motion
konka1机器人运动控制模块（Version 1.1）

本版本主要改动如下：
1. 修改了底盘运动模式，由速度模式改为位置模式
2. 修复了底盘初始位置的bug，现在每次调用规划器时，使用的是当前机器人坐标系，底盘位置为0，感知数据也应该是最新的
3. 修复了关节反馈实时性的bug，motion_generator现在的服务分为两个功能，第一个是查询关节状态，第二个是轨迹规划，需要在小脑中按顺序调用两次，中间间隔0.2秒以上
4. 小脑模块执行任务时由异步调用改为异步调用+同步等待，保证了下次服务开始前，上次服务已经结束，而且各个函数被解耦，代码复用率大大增加
5. 新增了release功能，该功能和pickup一样暂不完善，没有添加感知数据中的障碍物体素
6. 摄像头tf矩阵改为实时获取，每次抓取前不再需要调整摄像头
7. 修复了摄像头内参矩阵的bug
8. 添加了虚拟大脑模块，便于调试
9. 机器人配置文件存放到了motion_generator功能包中
10. 共享的消息类型kongka1_base中的ExecuteTrajectory.srv和GeneratePlan.srv添加到了gongga_interface中
11. 安装空间内添加了setup_env.bash文件，用于制定curobo路径，现在motion_generator模块可以使用ros2 run命令正常运行了
